# Project Agent Guidelines

This repository uses shared agent conventions stored in `.cursor/skills/` and `.cursor/rules/`. When working on files in this project, follow the guidelines defined there.

## Alloy Configuration

When editing or generating Grafana Alloy configuration files (`*.alloy`), you **must** read and follow the instructions in:

- [.cursor/skills/the-moll/SKILL.md](.cursor/skills/the-moll/SKILL.md)

These instructions cover required comment conventions, formatting standards, and quality checks for all Alloy config in this repository. Do not skip them.
